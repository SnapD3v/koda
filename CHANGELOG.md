## v0.2.2 — 2026-06-11

### Поиск
- Фильтрация результатов по типу: Все / Фильмы / Сериалы / Аниме
- Пагинация: кнопка "Загрузить ещё" при наличии следующей страницы
- Фильтрация выполняется на стороне клиента - учитываются все подтипы (foreign-movie, anime-serial и т.д.)

### Плеер
- Диалог "Продолжить с X:XX / С начала" перед запуском эпизода с сохранённой позицией
- Горячая клавиша `T` для переключения озвучки на экране деталей

### Скачивание
- Выбор озвучки прямо в окне загрузчика; при возврате назад выбранная озвучка синхронизируется с экраном деталей
- Убран выбор сезона — используется текущий сезон автоматически

### Сборка
- Кастомная иконка приложения (Windows - `.ico`, Linux AppImage - `.png`)
- Автообновлятор: загрузка и замена исполняемого файла с прогрессом и подтверждением
- Сборка под Linux переведена на формат AppImage

### Исправления
- При возврате на экран поиска позиция прокрутки восстанавливается корректно
- Исправлен порядок инициализации конфига в приложении

## v0.2.0 — 2026-06-03

### Что нового

#### Скачивание
- Прогресс каждого эпизода отображается в реальном времени (размер + таймкод)
- Пауза и возобновление загрузки в любой момент
- Системное уведомление рабочего стола по завершении

#### Плеер
- Автоматический переход к следующему эпизоду после просмотра (5 сек, отмена клавишей S)
- Интеграция с oscc: если скрипт установлен, mpv получает правильный заголовок для поиска субтитров

#### Библиотека
- Полнотекстовый поиск внутри папки
- Полоса прогресса просмотра у каждого эпизода
- Перетаскивание папок для изменения порядка

#### Интерфейс
- Главный экран с навигацией
- Экран настроек: токен, плеер, качество, папка загрузок
- Всплывающий список горячих клавиш по `?`
- История позиции прокрутки при возврате в поиск
- Очистка истории поиска

#### Сборка
- Portable Windows-билд с бандлом mpv + oscc
- Фоновая проверка обновлений при запуске
- Сборка под macOS и Linux
